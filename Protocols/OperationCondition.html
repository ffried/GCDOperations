OperationCondition

public protocol OperationCondition

A protocol for defining conditions that must be satisfied in order for an operation to begin execution.

  • name

    The name of the condition. This is will be passed as conditionName in ConditionErrors.

    Declaration

    Swift

    static var name: String { get }
  • isMutuallyExclusive

    Specifies whether multiple instances of the conditionalized operation may be executing simultaneously.

    Declaration

    Swift

    static var isMutuallyExclusive: Bool { get }
  • dependency(for:)

    Some conditions may have the ability to satisfy the condition if another operation is executed first. Use this method to return an operation that (for example) asks for permission to perform the operation

    Note

    Only a single operation may be returned as a dependency. If you find that you need to return multiple operations, then you should be expressing that as multiple conditions. Alternatively, you could return a single GroupOperation that executes multiple operations internally.

    Declaration

    Swift

    func dependency(for operation: Operation) -> Operation?

    Parameters

    operation

    The Operation to which the Condition has been added.

    Return Value

    An Operation, if a dependency should be automatically added. Otherwise, nil.
