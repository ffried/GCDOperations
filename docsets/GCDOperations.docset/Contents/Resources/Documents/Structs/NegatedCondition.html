NegatedCondition

public struct NegatedCondition<ConditionToNegate> : OperationCondition where ConditionToNegate : OperationCondition

A simple condition that negates the evaluation of another condition. This is useful (for example) if you want to only execute an operation if the network is NOT reachable.
